# THEBEAUTIFULGAME
Full Stack MERN Social Media App

### Introduction
This is a Full Stack MERN ( MongoDB, ExpressJs React and Node.js) Application.

###  Features
- User Authentication
- Create, Edit and Delete posts
- Search feature based on name and tags 
- Recommended posts section
- Comments section
